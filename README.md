# PlayerManager (Fabric 1.21.11)

Мод **только для клиента**. Позволяет смотреть “глазами” других сущностей.

## Управление

- `V` — включить/выключить режим (камера переключится на сущность под прицелом, или на ближайшую).
- `[` и `]` — переключить камеру на предыдущую/следующую ближайшую сущность (работает только когда режим включен).

## Сборка на своём ПК (Linux)

1) Поставь **Java 21**.

Проверка:

```bash
java -version
```

Должно быть что-то вроде `21.x`.

2) Открой терминал в папке проекта и собери:

```bash
cd "entity-cam-1.21.11"
./gradlew build
```

3) Готовый мод будет тут:

- `build/libs/playermanager-1.0.0.jar`

Закинь этот `.jar` в свою папку `mods`.

## Сборка на GitHub (чтобы “даже тупой понял”)

### 1) Загрузить проект на GitHub

1) Создай новый репозиторий на GitHub (например `playermanager`).
2) Закинь в него **все файлы** из папки `entity-cam-1.21.11`.

Если делаешь через git:

```bash
cd "entity-cam-1.21.11"
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin <ССЫЛКА_НА_ТВОЙ_РЕПОЗИТОРИЙ>
git push -u origin main
```

### 2) Запустить сборку

1) Открой вкладку **Actions** в репозитории.
2) Там появится workflow **Build**.
3) Нажми **Run workflow** (или просто сделай любой commit — сборка запустится сама).

### 3) Скачать готовый .jar

1) В Actions открой последний успешный запуск.
2) Внизу будет **Artifacts** → скачай `playermanager-jar`.
3) Внутри будет `.jar`, кидай его в `mods`.
